Saher Gerais Landing Page

This project is a personal landing page created by Saher Gerais to showcase web development skills and provide a contact point for potential services.
 
Features
 
Personal Introduction: A brief bio highlighting the developer's journey and growth in web development.

Skills Showcase: Highlights the ability to build functional and user-friendly interfaces using core web technologies.

Contact Form: A clean, simple form for users to provide their email address to get in touch.

Live Notifications: Includes a placeholder for dynamic Bootstrap alerts to provide user feedback.

Technologies Used

HTML5: For the semantic structure of the landing page.

CSS3: Linked via index.css for custom visual design and layout.

JavaScript: Linked via script.js to handle interactive features like alerts.

Bootstrap v5.3.0: Utilized for responsive layout and pre-built UI components.

Project Structure

index.html: The main entry point containing the page structure and content.

index.css: Custom stylesheet for the "graphics" main container and text styling.

script.js: JavaScript logic for the "Contact me" button and alerts.

bootstrap.min.css / bootstrap.bundle.min.js: Local Bootstrap framework files for styling and interactivity.

Setup

Clone or download the project files to your local environment.

Ensure that index.css, script.js, and the Bootstrap files are in the same directory as index.html.

Open index.html in any modern web browser to view the landing page.
